Features
=================================
Online Registration System will have the following features:
1. On line submission of application forms
2. On-line Quizzes for the first semester courses.  There should be minimum 10 questions(objective types with four options) When user leaves the test its result should be displayed at that moment. 
3. There should be a proper login system for the user where security is required. 
4. The web technologies to be used ; DHTML, JavaScript  and ASP
5. The proper validation should be done.
